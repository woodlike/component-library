# Text

## Text Sizes

Covers sizes from **s** to **xxxl**.
The Text component also has a `tag` prop that accepts a `p` or `div` in order to render the proper HTMLElement.
<Text tag="p" size="xl">
  I watched the storm, so beautiful yet terrific.
</Text>
<Text tag="p" size="l">
  I watched the storm, so beautiful yet terrific.
</Text>
<Text tag="p" size="m">
  I watched the storm, so beautiful yet terrific.
</Text>
<Text tag="p" size="s">
  I watched the storm, so beautiful yet terrific.
</Text>
