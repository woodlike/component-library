# Headings

## Props glossary

<Props of={Heading} />

## Heading Sizes

Covers sizes from **s** to **xxxl**.
The heading component also has a `tag` prop that accepts all six section levels according the spec.

<Playground>
  <Heading tag="h1" size="xxl">
    Healing Flowers Collection
  </Heading>
  <Heading tag="h2" size="xl">
    Healing Flowers Collection
  </Heading>
  <Heading tag="h3" size="l">
    Healing Flowers Collection
  </Heading>
  <Heading tag="h4" size="m">
    Healing Flowers Collection
  </Heading>
  <Heading tag="h5" size="s">
    Healing Flowers Collection
  </Heading>
</Playground>

## Heading Families

Covering three families as defined in the theme `heading` props.

- **display**
- **secondary**
- **campaign**

<Playground>
  <Heading tag="h2" size="xxl">
    Healing Flowers Collection
  </Heading>
  <Heading tag="h2" size="xxl" family="secondary">
    Healing Flowers Collection
  </Heading>
  <Heading tag="h2" size="xxl" family="campaign">
    Healing Flowers Collection
  </Heading>
</Playground>
