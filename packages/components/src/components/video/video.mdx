# Video

Video sizes are defined by their original sources. For woodlike we recommend the three following sizes for art directing videos on different devices.

## Props glossary

<Props of={Video.Media} />

## Large

Video dimension (9:16 | 1920 x 1080). Ideal for laptops and desktops.
It has no native controls, autoplay is on and it is muted.

<Playground>
  <Intersection
    options={{
      threshold: [0.1, 0.5, 1.0],
    }}
    render={data => (
      <IntersectionTarget ref={data.targetRef}>
        <Video.Media
          controls={false}
          autoPlay={true}
          loop={true}
          muted={true}
          sources={[
            {
              id: 'first-demo-video-mp4',
              src: '/public/woodlike-sustainable-swimwear-healing-flowers-large.mp4',
              type: 'video/mp4',
            },
            {
              id: 'first-demo-video-web',
              src: '/public/woodlike-sustainable-swimwear-healing-flowers-large.webm',
              type: 'video/webm',
            },
          ]}
        />
      </IntersectionTarget>
    )}
  />
</Playground>
